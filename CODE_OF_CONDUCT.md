# Contributor Covenant Code of Conduct

This project has adopted the [Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/) as its code of conduct.

All contributors, maintainers, and participants are expected to follow these standards to ensure a welcoming and inclusive community.

## Summary

We are committed to providing a friendly, safe, and welcoming environment for all, regardless of background or identity.

Please refer to the [full text of the Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/) for complete details on expected behavior, enforcement guidelines, and reporting procedures.

## Reporting

If you experience or witness unacceptable behavior, please report it by opening a GitHub issue or contacting the maintainers directly.
